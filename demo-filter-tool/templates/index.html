Talent Demographics (Instagram)

How to read index
  • An index shows how a talent's audience compares to the Instagram average for a given demographic.
  • Index values center on 1
  • Eg. A 1.5 means this talent performs 50% better than average in that demographic
  • Eg. A 0.6 means 40% worse (1 − 0.6 = 0.4).
  • Try clicking an index value to get the English Interpretation of the Index value!
Average Index Explainer
  • Average Index summarizes a talent's index values across all selected demographics into a single number
  • The Average index is centered at 1, just like the individual index values: 1.5 means the talent performs ~50% above typical overall; 0.6 means ~40% below typical overall.
  • We use a geometric (not arithmetic) mean because index values are ratios — a 2.0 and a 0.5 are equal-and-opposite swings, and only the geometric mean averages them back to exactly 1.
  • Caveat: a value near 1 doesn't always mean "no skew." It can also mean strong over-performance in one demographic is canceling out strong under-performance in another.
  • One-sentence Explainer for a Client:
    A single number representing how much this talent over or under-performs across the selected demographics.
